# Remote Work Security Policy

## Purpose
Ensure secure access and use of company systems when working outside the office.

## Scope
Applies to all employees connecting remotely via VPN, home networks, or public Wi-Fi.

## Requirements
- **VPN:** Mandatory connection for all remote access.
- **Device Security:** Company laptops must have disk encryption, antivirus, and automatic updates enabled.
- **Wi-Fi:** Avoid public Wi-Fi; if necessary, use company-approved VPN.
- **Password & MFA:** Strong passwords plus MFA required for cloud and internal systems.
- **Data Handling:** No storing company files on personal devices or unencrypted USB drives.

- **Remote Users:** Follow all requirements; report security incidents immediately.
- **IT Support:** Provide VPN setup and device configuration.
- **Management:** Monitor remote work compliance.
