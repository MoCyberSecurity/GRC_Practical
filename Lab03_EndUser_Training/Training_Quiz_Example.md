# End-User Security Quiz Example

## Purpose
Test employee understanding of cybersecurity awareness.

### Sample Questions

1. What makes a password strong?
   - a) 123456
   - b) Unique phrase + symbols
   - c) Your pet’s name
   - **Answer:** b

2. How should you handle a suspicious email?
   - a) Delete immediately
   - b) Click the link to verify
   - c) Report to IT/security
   - **Answer:** c

3. What is MFA?
   - a) Multi-Factor Authentication
   - b) Mainframe Access
   - c) Manual File Approval
   - **Answer:** a

### Scoring & Follow-up
- 80–100%: Excellent
- 60–79%: Needs improvement – follow-up training
- <60%: Mandatory refresher session
